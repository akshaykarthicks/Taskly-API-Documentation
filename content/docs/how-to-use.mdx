## Getting Started with Taskly.AI

This guide will walk you through everything you need to know to use Taskly.AI effectively.

## Access Taskly.AI

Visit the live application: [https://taskly-ai-five.vercel.app/tasks](https://taskly-ai-five.vercel.app/tasks)

## Creating Your First Task

<Steps>

<Step>

### Click "Create New Task"

Look for the button to create a new task on the main interface.

![Create Task Button](/images/create-task.png)

</Step>

<Step>

### Fill in Task Details

**Title** (Required)
- Enter a clear, descriptive title
- Example: "Build crew AI project for food crew"

**Description** (Optional but recommended)
- Add detailed information about your project
- Include goals, requirements, and context
- More details = better AI suggestions

**Priority**
- Choose: High, Medium, or Low
- High: Urgent tasks
- Medium: Important tasks
- Low: Can wait

**Status**
- To Do: Not started yet
- In Progress: Currently working on
- Completed: Finished

**Due Date**
- Set a deadline for your task
- Use the date picker
- Optional but helps with planning

</Step>

<Step>

### Generate AI Subtasks (Optional)

This is where the magic happens!

1. After filling in title and description
2. Click "Generate Subtasks" button
3. Wait a few seconds for AI to analyze
4. Review the generated subtasks
5. Edit or customize as needed

<Callout type="tip">
For best AI results, include timeline, team size, and specific requirements in your description!
</Callout>

</Step>

<Step>

### Save Your Task

Click "Create Task" button to save your task with all the details and subtasks.

</Step>

</Steps>

## Using AI Effectively

### Write Good Task Descriptions

**Basic Description:**
```
Build a website
```

**Better Description:**
```
Build a company website with modern design, 
mobile responsive, contact form, and blog section
```

**Best Description:**
```
Build a company website using React and Tailwind CSS.
Features needed:
- Modern, professional design
- Fully mobile responsive
- Contact form with email integration
- Blog section with CMS
- SEO optimization
- Fast loading speed

Timeline: 3 weeks
Team: 2 developers, 1 designer
```

### Include Key Information

For best AI results, mention:
- **What**: Project goal and deliverables
- **How**: Technologies or methods
- **When**: Timeline or deadline
- **Who**: Team size or resources
- **Why**: Context or requirements

## Tips for Maximum Productivity

### 1. Use Priorities Wisely

- **High**: Only for truly urgent tasks
- **Medium**: Most of your tasks
- **Low**: Nice-to-have items

### 2. Set Realistic Due Dates

- Don't overcommit
- Add buffer time
- Consider dependencies

### 3. Break Down Large Projects

- Use AI to generate subtasks
- Make subtasks actionable
- One clear action per subtask

### 4. Review Your Dashboard Daily

- Check what's due soon
- Focus on high priority
- Track your progress

### 5. Update Status Regularly

- Move tasks to "In Progress" when starting
- Mark completed tasks promptly
- Keep your dashboard accurate

### 6. Use Detailed Descriptions

- More context = better AI suggestions
- Include specific requirements
- Mention constraints or limitations

### 7. Customize AI Suggestions

- AI provides a starting point
- Edit subtasks to match your workflow
- Add or remove steps as needed
