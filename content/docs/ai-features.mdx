### Include Key Information

For best AI results, mention:

1. **Project Goal**: What you want to achieve
2. **Scope**: What's included and excluded
3. **Timeline**: Desired completion time
4. **Resources**: Team size, tools, budget
5. **Constraints**: Limitations or requirements
6. **Deliverables**: Expected outcomes

### JavaScript Example

```javascript
async function generateSubtasks(taskId, taskDescription) {
  const response = await fetch(`/api/tasks/${taskId}/generate-subtasks`, {
    method: 'POST',
    headers: {
      'Authorization': `Token ${authToken}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({
      task_id: taskId,
      description: taskDescription,
      context: {
        timeline: '2 weeks',
        team_size: 3
      }
    })
  });

  const data = await response.json();
  return data.subtasks;
}
```

### Python Example

```python
import requests

def generate_subtasks(task_id, description, timeline="2 weeks"):
    url = f"http://localhost:8000/api/tasks/{task_id}/generate-subtasks"
    headers = {
        "Authorization": f"Token {auth_token}",
        "Content-Type": "application/json"
    }
    payload = {
        "task_id": task_id,
        "description": description,
        "context": {
            "timeline": timeline,
            "team_size": 2
        }
    }
    
    response = requests.post(url, json=payload, headers=headers)
    return response.json()["subtasks"]
```
